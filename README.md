## Configuration
You can adjust some values in the `.ini` file :
- 🧍 Use story mode characters as suspects or random characters.
- ☑️ Enable/Disable individual callouts.
- 🏠 Force the use of interiors for callouts.

## Known issues
- Prologue : The pathfinding may break.
- Pack Man : The Monroe may move around a bit when detached from the trailer.
- The Big Score - Obvious : The suspects might teleport along the way if they take too long to move.
- The Third Way : The foundry interior is not detected correctly. The interior version of the callout is only selected if "forceInteriorsEnabled" is set to True.
- Grass Roots - The Pickup : The Duneloader may spawn with objects in the back that conflict with the box.
